# HappyBirthdayBot

Telegram-бот для организации групповых поздравлений с днём рождения.

## Пользовательские команды

| Команда | Описание |
|---|---|
| `/start` | Приветствие и список доступных команд |
| `/list` | Все участники программы с датами дней рождения |
| `/reminders` | 3 ближайших именинника с количеством дней до события |
| `/my_birthday` | Зарегистрироваться в программе или обновить свою дату рождения. Формат ввода: `Вася Пупкин, 25.03` или `Вася Пупкин, 25 марта` |
| `/wishlist` | Посмотреть и изменить свой вишлист. Также позволяет посмотреть вишлисты других участников |
| `/exit` | Выйти из программы — бот перестанет поздравлять с днём рождения |

## Автоматические напоминания

Бот сам рассылает уведомления в основной чат по расписанию (cron):

| Триггер | Действие |
|---|---|
| За 30 дней | Сообщение с напоминанием о предстоящем дне рождения, ссылкой на чат подарка и вишлистом именинника |
| За 14 дней | Повторное напоминание со ссылкой на чат подарка |
| В день рождения | Поздравление с днём рождения |

## Админские команды

Работают **только** из admin chat (личка администратора бота).

| Команда | Описание |
|---|---|
| `/remind` | Немедленно запустить задачу рассылки напоминаний, не дожидаясь cron |
| `/exec <sql>` | Выполнить произвольный SQL-запрос к базе данных SQLite |
